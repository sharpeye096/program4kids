# 第二章：编程基础概念

## 2.1 变量：数据的容器

### 什么是变量？
变量就像是给数据起的名字标签🏷️，或者是一个小盒子📦，用来存放东西。

### 如何创建变量？
```python
# 给数字起名字
年龄 = 10
分数 = 98.5

# 给文字起名字  
名字 = "小明"
学校 = "阳光小学"

# 给真假值起名字
是学生 = True
喜欢编程 = False
```

### 变量命名规则：
1. **可以包含**：字母、数字、下划线 `_`
2. **不能以数字开头**：`1name` ❌，`name1` ✅
3. **区分大小写**：`age` 和 `Age` 是不同的变量
4. **不能使用关键字**：比如 `print`、`if` 等

### 使用变量：
```python
名字 = "小红"
年龄 = 9
print(名字 + "今年" + str(年龄) + "岁了")
# 输出：小红今年9岁了
```

## 2.2 数据类型：数字、文字、真假值

### 数字类型
```python
# 整数 - 没有小数点的数字
年龄 = 10          # int类型
班级人数 = 45      # int类型

# 浮点数 - 有小数点的数字  
身高 = 1.45        # float类型
体重 = 35.5        # float类型
温度 = 36.8        # float类型
```

### 文字类型（字符串）
```python
# 用单引号或双引号包围
名字 = '小明'       # str类型
学校 = "希望小学"    # str类型
格言 = '好好学习，天天向上'  # str类型

# 特殊字符
换行 = "第一行\n第二行"  # \n表示换行
制表符 = "姓名\t年龄"    # \t表示制表符（Tab）
```

### 真假值类型（布尔值）
```python
是晴天 = True      # bool类型，表示真
下雨了 = False     # bool类型，表示假

# 比较运算得到布尔值
结果1 = 10 > 5     # True，10大于5
结果2 = 3 == 4     # False，3不等于4
```

### 查看数据类型：
```python
print(type(10))        # <class 'int'>
print(type(3.14))      # <class 'float'>
print(type("hello"))   # <class 'str'>
print(type(True))      # <class 'bool'>
```

## 2.3 基本运算：加减乘除

### 算术运算
```python
# 加法
和 = 5 + 3        # 8

# 减法  
差 = 10 - 4       # 6

# 乘法
积 = 6 * 7        # 42

# 除法
商 = 15 / 3       # 5.0（注意结果是浮点数）

# 整除（去掉小数部分）
整除结果 = 17 // 5   # 3

# 取余（求余数）
余数 = 17 % 5      # 2

# 幂运算
平方 = 5 ** 2      # 25
立方 = 2 ** 3      # 8
```

### 字符串运算
```python
# 字符串拼接
问候 = "你好" + "！"          # "你好！"
全名 = "张" + "小明"          # "张小明"

# 字符串重复
星星 = "⭐" * 5              # "⭐⭐⭐⭐⭐"
分隔线 = "-" * 20            # "--------------------"

# 注意：数字和字符串不能直接相加
# print("年龄：" + 10)       # ❌ 错误！
print("年龄：" + str(10))    # ✅ 正确！
```

## 2.4 输入和输出：与程序对话

### 输出：print()函数
```python
# 基本输出
print("Hello World!")

# 输出多个值
print("姓名：", "小明", "年龄：", 10)

# 格式化输出
名字 = "小红"
年龄 = 9
print(f"{名字}今年{年龄}岁了")  # 小红今年9岁了

# 输出到一行（不换行）
print("正在加载", end="")
print("...", end="")
print("完成！")
# 输出：正在加载...完成！
```

### 输入：input()函数
```python
# 获取用户输入
名字 = input("请输入你的名字：")
年龄 = input("请输入你的年龄：")

print(f"你好{名字}，你{年龄}岁了！")

# 注意：input()返回的是字符串
# 如果需要数字，要转换类型
年龄数字 = int(input("请输入年龄："))
明年年龄 = 年龄数字 + 1
print(f"明年你就{明年年龄}岁了")
```

### 类型转换
```python
# 字符串转数字
数字1 = int("10")      # 10（整数）
数字2 = float("3.14")  # 3.14（浮点数）

# 数字转字符串
文字1 = str(10)        # "10"
文字2 = str(3.14)      # "3.14"

# 布尔值转换
结果1 = bool(1)        # True
结果2 = bool(0)        # False
结果3 = bool("hello")  # True
结果4 = bool("")       # False
```

## 2.5 小练习

### 练习1：个人信息收集
```python
# 收集用户信息并显示

年龄 = int(input("你几岁了？"))

print(f"姓名：{名字}")
print(f"年龄：{年龄}")
print(f"学校：{学校}")
print("=" * 20)
```

### 练习2：数学小助手
```python
# 帮助计算数学题
数字1 = float(input("请输入第一个数字："))
数字2 = float(input("请输入第二个数字："))

print(f"\n计算结果：")
print(f"{数字1} + {数字2} = {数字1 + 数字2}")
print(f"{数字1} - {数字2} = {数字1 - 数字2}")
print(f"{数字1} × {数字2} = {数字1 * 数字2}")
print(f"{数字1} ÷ {数字2} = {数字1 / 数字2}")
```

### 练习3：创意作品
