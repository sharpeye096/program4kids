Python 趣味数学实验室：课题建议核心理念：不讲枯燥的公式，只玩“有用的规律”。适用工具：Python (基础运算 + Turtle + Random)课题 1：时钟世界的数学 (模运算 Modular Arithmetic) 🕒——“为什么 10 + 4 = 2 ？”这是您提到的模加法，也是计算机科学中最重要的概念之一（密码学的基础）。1. 数学概念时钟算术：在一个 12 小时的时钟上，10 点过 4 个小时是几点？是 2 点，不是 14 点。取余数：$14 \div 12 = 1 \dots 2$。这个“2”就是模运算的结果。记号：$14 \pmod{12} = 2$。2. Python 实践运算符：介绍 % (取模/取余)。代码实验：print(14 % 12)  # 输出 2
print(100 % 7)  # 100天后是星期几？
3. 实战项目：转圈圈的颜色玩法：我们有一个颜色列表 colors = ['red', 'blue', 'green'] (长度是3)。问题：如果在循环里 i 增加到 100，怎么防止索引报错（IndexError）？魔法：用 i % 3！无论 i 多大，结果永远在 0, 1, 2 之间循环。Turtle 画图：colors = ['red', 'yellow', 'blue', 'purple']
for i in range(100):
    t.pencolor(colors[i % 4]) # 永远循环取这4个颜色
    t.forward(i)
    t.left(91)
(这会画出一个非常漂亮的彩色螺旋，孩子会瞬间明白模运算的“循环”特性)课题 2：外星人的手指 (二进制 Binary) 👽——“如果外星人只有 2 根手指怎么数数？”帮助孩子理解计算机为什么只认识 0 和 1。1. 数学概念位值原理：我们用十进制是因为有10根手指。二进制：逢二进一。0110 (代表2)11 (代表3)100 (代表4)2. Python 实践指令：bin() 函数。print(bin(5)) # 输出 '0b101'

Python 实践工具：random 模块，勾股定理 ($x^2 + y^2 < r^2$)。代码逻辑：随机生成一个点 $(x, y)$。计算它到圆心的距离（复习距离公式）。如果在圆内，计数器 +1。扔 1000 次，看有多少次在圆内。3. 实战项目：寻找 $\pi$可视化：画一个正方形框。快速随机打点（Turtle dot()）。圆内的点画红色，圆外的点画蓝色。最后算一下：$\text{红点数} \div \text{总点数} \times 4$，看看结果是不是接近 3.14？(这个过程非常有视觉冲击力，随着点越来越多，数值会越来越准)课题 4：大自然的密码 (斐波那契数列 Fibonacci) 🐚——“为什么花瓣的数量总是 3, 5, 8, 13 ？”结合 L6 列表和 L3 循环的最佳练习。1. 数学概念数列规则：前两个数加起来等于第三个数。1, 1, 2, 3, 5, 8, 13, 21...2.
